## Experiment Title

Verification of algebraic and bitwise identities using Python

## Aim

To write a Python program to verify basic algebraic (binomial and cubic) identities and related bitwise operator identities.

## Algorithm

1. Initialize two integer variables `a` and `b` with constant values.
2. Compute and compare the left-hand side and right-hand side of different algebraic identities using arithmetic operators.
3. Print the Boolean result (`True` or `False`) of each comparison.
4. Compute and compare the left-hand side and right-hand side of different bitwise identities using bitwise operators (`|`, `&`, `^`).
5. Print the Boolean result (`True` or `False`) of each bitwise comparison.
6. Observe the output to confirm the correctness of the identities.
