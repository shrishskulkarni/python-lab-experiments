## Experiment Title

Experiment 9

## Aim

To write a program using Pandas to create and display a DataFrame and use Matplotlib to plot the data.

## Algorithm

1. Import `pandas` as `pd` and `matplotlib.pyplot` as `plt`.
2. Create two lists: `SL` and `Name`.
3. Create a DataFrame using `pd.DataFrame()` with columns `SL` and `Name`.
4. Print the DataFrame.
5. Plot `Name` on the x-axis and `SL` on the y-axis using `plt.plot()`.
6. Add title and axis labels.
7. Display the plot using `plt.show()`.

## Output

Sample output (DataFrame):

```text
    SL     Name
0  101     Bran
1  102    Jamie
2  103  Baalish
3  104      Jon
4  105     Arya
```

And a line plot window titled `GoT` will be displayed.
